Codevs 1220 数字三角形

【题目描述】

有一个数字三角形，从顶部出发，在每一结点可以选择向左走或得向右走，一直走到底层，要求找出一条路径，使路径上的值最大。

【输入描述】

第一行是数塔层数$n(1 \le n \le 100)$。

第二行起，按数塔图形，有一个或多个的整数，表示该层节点的值，共有$n$行。

【输出描述】

输出最大值。

【题解】

设$f(i, j)$表示从$(i, j)$出发向下走的路径上的最大值，则转移为
$$
f(i, j) = \max(f(i + 1, j), f(i + 1, j + 1)) + a[i][j]
$$
边界为$f(n, j) = a[n][j], j \in [1, n]$
